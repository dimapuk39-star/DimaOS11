# DimaOS 11

Интерактивная браузерная операционная система в стиле Windows 11.

## Быстрый запуск без npm

Дважды щёлкните файл `START-DimaOS.cmd`. Откроется браузер с DimaOS 11.

Окно запуска должно оставаться открытым. Для остановки нажмите в нём `Ctrl+C`.

## Обычный запуск для разработки

```bash
npm install
npm run dev
```

После запуска откройте адрес, который покажет Vite (обычно `http://localhost:5173`).

## Что работает

- меню «Пуск» и поиск приложений;
- перетаскиваемые окна, сворачивание, разворачивание и закрытие;
- «Проводник», Dima Edge, Dima Store, «Фотографии», «Блокнот» и терминал;
- «Параметры» с навигацией и разделом «Система → О системе»;
- характеристики процессора, видеокарты, памяти и накопителя;
- выбор обоев в разделе «Персонализация»;
- быстрые настройки, календарь и живые часы.
- полноценный виртуальный Проводник с локальным сохранением файлов и папок;
- создание, импорт, редактирование, поиск, сортировка, копирование и корзина;
- калькулятор с памятью и историей вычислений;
- диспетчер задач с процессами и живым графиком производительности;
- локальный помощник Dima AI, запускающий приложения командами;
- Dima Edge с адресной строкой, историей и переходом на настоящие сайты.
- Dima Browser с вкладками, журналом, закладками и восстановлением сессии;
- Dima Player с локальной медиатекой, аудио, видео, очередью и Picture-in-Picture;
- сохранение медиафайлов и пользовательских обоев через IndexedDB;
- сохранение положения окон, яркости, быстрых настроек и персонализации;
- приветственный экран `Hello, Дмитрий` после загрузки системы.
- реальные характеристики компьютера через локальную системную службу;
- автоматический браузерный режим при недоступности системной службы.

Это самостоятельный интерфейс DimaOS 11 с оригинальными компонентами и без фирменных логотипов Microsoft.

## Dima AI

Dima AI работает в трех режимах:

- встроенный режим — системные команды без Интернета;
- OpenAI API — полноценный разговорный помощник;
- Ollama — локальная модель на компьютере.

Для OpenAI запустите `SETUP-DimaAI.cmd`, вставьте API-ключ и перезапустите DimaOS. Ключ хранится в пользовательской переменной окружения Windows и не передается клиентскому JavaScript.

Для Ollama запустите Ollama на `http://127.0.0.1:11434`, затем выберите Ollama в настройках Dima AI.
